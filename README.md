# 概要

Emacs で字を大きく表示する program です．

# 使ひ方

0. 大きくしたい文字列を region にしてください．

1. `Sakurahasi-cmd` を實行してください．

2. 字が別 window に大きく表示されます．

3. Window はお好みの方法 ( `popwin.el`, `winner` 等 ) で消してください．

# 名前の由來

私の故郷である靜岡市にある靜岡鐵道の櫻橋 ( さくらはし ) 驛から名前をもらつてゐます．
